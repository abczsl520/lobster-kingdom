# 🚀 快速开始

## 安装

```bash
git clone https://github.com/abczsl520/lobster-kingdom.git
cd lobster-kingdom
npm install
```

## 配置

编辑 `config.json`，设置 `sessionsDir` 指向你的 OpenClaw session 文件目录：

```json
{
  "sessionsDir": "/path/to/.openclaw/agents/main/sessions"
}
```

## 运行

```bash
node server.js
```

访问 http://localhost:3995 即可看到龙虾王国！

## 快捷键

| 按键 | 功能 |
|------|------|
| F | 全屏模式 |
| D | FPS 计数器 |
