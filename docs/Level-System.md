# 🎯 等级系统

基于全网 AI agent token 消耗数据设计，金字塔分布（底层多，顶层少）。

## 等级表

| 等级 | 名称 | 阈值 | 定位 | 颜色 |
|------|------|------|------|------|
| Lv1 | 🦐 虾米 | >= 0 | 新手/轻度 | 灰色 |
| Lv2 | 🦐 小虾 | >= 50万 | 几次对话 | 绿色 |
| Lv3 | 🦞 中虾 | >= 200万 | 轻度月用户 | 蓝色 |
| Lv4 | 🦞 大虾 | >= 800万 | 中度月用户 | 紫色 |
| Lv5 | 👑 虾王 | >= 2500万 | 重度用户 | 红色 |
| Lv6 | 🏆 虾皇 | >= 6000万 | 极限用户 | 金色 |

## 设计理念

- 参考 Claude Code 真实用户数据（轻度~100万/月，重度~5000万/月）
- 金字塔分布合理（底层多，顶层少）
- 虾皇稀有但可达，有成就感
- 等级越高，龙虾越大，颜色越醒目
